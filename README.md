## 📌 Overview

**AlFajr Offer Printing System** is a browser-based application designed to create professional promotional offer sheets for stores and supermarkets.

The application provides a live preview while allowing complete customization of the printed layout, including text, fonts, colors, backgrounds, borders, paper orientation, and print size.

No installation or backend server is required.

---

## ✨ Features

- 📝 Three editable text sections
- 🔤 Multiple Arabic font families
- 📏 Adjustable font sizes
- 🎨 Color presets for every text section
- **Bold**, *Italic*, and Underline formatting
- 🖼️ Default or custom background image
- 🌫️ Adjustable background opacity
- 🖨️ A4 Portrait & Landscape printing
- 📄 Full-page printing
- ✂️ Two offers per page
- 📑 Single half-page printing
- 🖼️ Customizable borders (style, width, color)
- 👀 Real-time preview
- 📱 Responsive interface for desktop and mobile
- ⚡ Pure HTML, CSS & JavaScript (No frameworks)

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/alfajr-offer-printing.git
```

Open the project folder:

```bash
cd alfajr-offer-printing
```

Run the application by opening:

```text
index.html
```

Or use **Live Server** in Visual Studio Code.
